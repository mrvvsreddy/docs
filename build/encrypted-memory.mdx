title: "Encrypted Long-term Memory"
description: "How Maedo agents remember and personalize user experiences."
---

Maedo agents aren't just one-off responders. They possess encrypted, long-term memory that allows them to build relationships with your customers over time.

### Persistent Context
Agents remember preferences, past purchases, and specific user goals across sessions. This allows for seamless follow-ups and highly personalized shopping experiences.

### Privacy First
All memory data is encrypted at rest and in transit, ensuring that personalized experiences don't come at the cost of security.
